Dockerized Mindspore
-------------------

# Current version

* ubuntu 18.04
* python 3.7
* mindspore 0.2.0



# Installation
```shell script
docker pull yarenty/mindspore
```

# Run
```shell script
docker run -it yarenty/mindspore
```

You should be in /opt directory, and you should be able to run:
```shell script
python3.7 test_mul.py
```

Output:
```text
[ 4. 10. 18.]
```
